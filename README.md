## What is this repo all about?

This repo contains collections for my [pg-finder](https://github.com/maninageswar/pg-finder) project.

## Rules to be followed if you are contributing to this repo

- Do not edit any existing file in this repo always create a new file.
- Please follow the `snake_case` for naming the files like `pg_properties_collection_schema_30_april_26.json`
- (IMPORTANT) If you have added a new file in this repo, please make sure to update the [README.md](https://github.com/maninageswar/pg-finder/blob/main/README.md) file of [pg-finder](https://github.com/maninageswar/pg-finder) repo.
